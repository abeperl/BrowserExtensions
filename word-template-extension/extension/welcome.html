Smart Data Extraction

Template Processing

One-Click Generation

Privacy First

All processing happens locally on your computer using native messaging. Your data never leaves your machine, ensuring complete privacy and security.

Getting Started

Follow these simple steps to set up and start using the extension

1

Install the Native Host

Run the installation script as Administrator in the native-host folder to enable communication between the browser and Word.

2

Add Your Templates

Place your Word templates in the Templates folder. Use placeholders like {{EMAIL}}, {{PHONE}}, {{DATE}} in your documents.

3

Extract and Generate

Visit any webpage, click the extension icon, extract data, select a template, and generate your document!
