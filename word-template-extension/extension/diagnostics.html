Troubleshooting Guide

Common issues and their solutions

"Native messaging host not found" Error

Solution:

  1. Run install.bat as Administrator in the native-host folder
  2. Update the extension ID using: .\update-manifest.ps1 -ExtensionId 'your-extension-id'
  3. Restart Chrome/Edge completely

"Extension ID mismatch" Error

Solution:

  1. Copy your extension ID from chrome://extensions/
  2. Run: .\update-manifest.ps1 -ExtensionId 'copied-extension-id'
  3. Run: install.bat as Administrator
  4. Restart browser

Python or Dependencies Issues

Solution:

  1. Install Python 3.7+ from python.org
  2. Run: pip install python-docx
  3. Run: diagnose.ps1 to verify setup

Templates Not Loading

Solution:

  1. Check that templates are in Documents\Templates\
  2. Ensure templates are .docx format
  3. Verify folder permissions
