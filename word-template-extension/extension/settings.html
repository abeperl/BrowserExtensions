Data Extraction

Configure how data is extracted from web pages

Format: fieldName:/regex/flags (one per line)
Examples:
• productIds:/PRD-\d{4,6}/g - Matches PRD-1234, PRD-123456
• orderNumbers:/ORD[A-Z0-9]{8}/gi - Matches ORDABC12345 (case insensitive)
• serialNumbers:/SN\d{10}/g - Matches SN1234567890
• licenseKeys:/[A-Z0-9]{4}-[A-Z0-9]{4}-[A-Z0-9]{4}/g - Matches ABCD-1234-EFGH
